# Population_Group_Wise_Deposit_Analysis
Excel_PowerBI_Project
## Project Overview

This project analyzes deposit patterns across different population groups using Power BI. The objective is to understand how deposits are distributed among various demographic segments and identify key contributors to overall deposits. The project demonstrates the use of data preprocessing, data modeling, visualization, and business intelligence techniques to generate actionable insights.

The raw dataset was first cleaned and prepared using Microsoft Excel before being imported into Power BI for analysis and dashboard development.
